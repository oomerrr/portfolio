My Projects

Project 1: Attendance Tracking System

A computer-vision and machine-learning–based application that automatically takes student attendance using OpenCV. Faces are detected and recognized in real time, and attendance records are exported directly into an Excel sheet. This solution streamlined classroom check-ins and reduced manual errors to nearly zero.

Project 2: Real Estate Value Predictor

An AI-driven regression project from an online course where multiple models (Linear Regression, Decision Trees, Random Forests, and Gradient Boosting) were trained and tested on housing-market data. Achieved sub-5% mean absolute error on the test set by tuning hyperparameters and engineering features like location indices and property age.

Project 3: Airplane Ticket Management System

A basic Python application with a graphical interface (Tkinter) to search, book, and cancel flight tickets—created for an introductory Python course. Later extended into a full database-backed system (sql, ~1,2000 records) to manage flights, passengers, and bookings, improving data persistence and query performance.

Project 4: Farm Land Management System

A capstone for Software Engineering: built a prototype web app to track crop rotations, irrigation schedules, and land parcels. Delivered full UML diagrams, requirements spec, design documentation, and user manual. Demonstrated end-to-end SDLC practices from requirements gathering through deployment.

Project 5: Python OOP Music Player

A Spotify-inspired desktop player built using Python’s object-oriented principles and a Tkinter GUI. Supports playlists, play/pause/skip controls, and dynamic library management. This was developed as part of an OOP-focused Python course, demonstrating clean class hierarchies and event-driven UI design.
